# NariBot Prompts
**Vernacular AI Logic for Livelihood Navigation**

This repository contains the system behavior logic and prompt templates for NariBot. 

### Overview
NariBot's AI Mentor is designed to provide brief, actionable advice to women job seekers and micro-entrepreneurs in India. It focuses on:
* **Vernacular Accessibility:** Support for Hindi, Kannada, Marathi, and Tamil.
* **Domain Focus:** Government schemes (Yojana), local business tips, and home-based work.
* **Channel Optimization:** Responses formatted for USSD (feature phones) and SMS.

### Logic Structure
The prompts follow a strict "Safety-First" framework to ensure advice stays within the livelihood and government scheme domains.
